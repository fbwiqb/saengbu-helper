# 생기부 입력 도우미 (saengbu-helper)

선생님이 학기별 생활기록부를 **입력·검토**할 때 돕는 데스크톱 도구. (AI가 대신 써주는 도구가 아님 — 입력 보조)

## 무엇을 도와주나
- **실시간 바이트 검사** — 영역별 한도(자율·자치/진로 1,500 · 행동특성 900 · 세특 1,500 · 동아리 1,500)
- **금지어 하이라이트** — 대학명·수상·논문·어학시험 등 기재 금지 표현
- **형식 검증** — 종결어미(~함/~임/~음/~됨), 독서 형식
- **반 전체 중복 방지** — 같은 행사 표현 사용 현황 + 유사도 경고
- **대시보드** — 학생×영역 작성 상태·진행률
- **품질 자가점검** — 경험·생각·성장·진정성 4기준

## 데이터
- 모든 학생 데이터는 **본인 PC(로컬 SQLite)에만** 저장됨. 외부 전송 없음.
- 학생 명단은 `import-roster`로 직접 등록(예시: `data/roster.sample.json`).

## 설치 / 실행
- 배포본: Releases의 설치 파일(.exe) 다운로드 → 실행. 새 버전은 **자동 업데이트**.
- 개발: `npm install` → `npm start` → http://localhost:5870

## 면책
